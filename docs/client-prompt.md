# Other Version Prompt
```
Design a complete high-end music streaming application UI system.

The product benchmarks QQ Music, but with a differentiated emotional direction:
warm sunset orange theme, cozy, night-friendly, soft glow atmosphere, premium and immersive.

Platforms:
- Mobile (primary)
- Tablet
- Desktop
- Android TV (10-foot experience, remote focus navigation)

Design System Requirements:
- Primary color: sunset orange (#FF7A45)
- Deep warm brown dark background (#1C1410)
- Soft glow orange accent lighting
- Large rounded corners (16–20px)
- Glassmorphism mini player
- Subtle gradient overlays
- Minimal but emotional
- 8pt grid system
- High-end product quality (Apple Music level)
- Consistent spacing and typography hierarchy
- Smooth motion concept (250ms transitions)
- Focus ring orange glow for TV

Core Pages to Generate:
1. Login page (phone + SMS verification)
2. Home page (carousel banner, recommended playlists, daily songs, new songs, new albums)
3. Search page (hot keywords, tabs: Songs / Artists / Albums / MV)
4. Library page (Liked songs, Playlists, History, real-time sync)
5. Full screen Player page (rotating album art, dynamic blurred background, synchronized lyrics)
6. Playlist detail page
7. Artist list page (filter by language, genre, gender)
8. Artist detail page
9. Album detail page
10. Ranking page (multiple charts with up/down indicators)
11. Radio page (personal FM style)
12. MV list page (grid layout, 16:9 thumbnails)
13. MV player page (fullscreen cinematic player)
14. Queue panel (draggable reorder)
15. Lyrics immersive mode

Interaction Requirements:
- Hover glow on cards (desktop)
- Focus scaling + orange ring (TV)
- Smooth expandable mini player
- Parallax header on detail pages
- Dynamic background based on album color
- Real-time favorite sync indicator
- Offline banner state

Visual Tone:
- Emotional
- Warm
- Cozy
- Premium
- Cinematic
- Immersive
- Not tech-green
- Not cold
- More human and intimate than QQ Music

Output:
Generate a complete UI mockup system with consistent visual language,
including mobile and desktop variations,
high resolution,
Figma-style layout,
clean product design,
professional streaming app interface.
```
